# I.このプロジェクトは、github アクションを使用して OpenWrt/LEDE システムをオンラインでコンパイルします

# II.説明書
## 1.このプロジェクトをフォークする
## 2.「アクション」をクリックし、「OpenWrt/LEDE システムのコンパイル」を選択し、「ワークフローの実行」をクリックして、ポップアップサブタブで「ワークフローの実行」をクリックします。
## 3.カスタマイズされたコンポーネントが必要な場合は、OpenWrt/LEDE プロジェクトのソース コードをダウンロードして自分で編集し、生成された `.config` ファイルを config ディレクトリに追加してください。
## 4.ファイルを変更する場合は、ワークフロー`main-n/a.yml`の14行目のCONFIG_FILE以降のファイル名を変更する必要があります。
## 5.このプロジェクトはクロスプラットフォーム コンパイルをサポートしています。ワークフローの下にある `main-n/a.yml` ファイルをコピーし、名前を変更します。対応するプラットフォームの `.config` ファイルを追加し、操作 3 と 4 を実行します。
